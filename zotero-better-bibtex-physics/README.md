# Zotero + Better BibTeX (Physics / LaTeX)

A minimal, stable Zotero configuration for generating clean APS/PRL-style
BibLaTeX output for physics theses and long LaTeX manuscripts.

---

## Guarantees
- Stable, human-readable citation keys
- Automatic BibLaTeX export
- No local file paths
- No abstracts, keywords, URLs, or access dates
- DOI-centric references
